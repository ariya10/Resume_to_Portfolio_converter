## Tips & Tricks

### 💡 Pro Tips

1. **Start with a Category**
   - Use category filters to narrow down templates
   - Browse within your target audience

2. **Preview Multiple Templates**
   - Don't settle for the first one
   - Compare 3-5 templates before deciding

3. **Use Complementary Colors**
   - Pick a template with colors that match your brand
   - Fine-tune if needed

4. **Keep Typography Professional**
   - Use serif fonts for headings in executive templates
   - Use sans-serif for body text for readability

5. **Less is More**
   - Don't enable all effects at once
   - Choose 2-3 visual enhancements maximum
   - Let content be the star

6. **Test Responsiveness**
   - Switch between desktop/tablet/mobile views
   - Ensure layout works on all devices

7. **Use High-Quality Profile Image**
   - Upload a professional headshot
   - Good lighting and clear focus
   - Square or circular images work best

### ⚠️ Common Mistakes to Avoid

- Don't overload with colors (stick to 3-4 main colors)
- Don't use too many different fonts (max 2-3)
- Don't enable all animation effects (choose your best)
- Don't forget to proofread before exporting
- Don't hide important sections (always show experience/skills)

## Export Guide

### PDF Export
1. Click **Export** → **Print / PDF**
2. Browser print dialog opens
3. Choose "Save as PDF"
4. Customize print settings if needed
5. Download and share

**Best for**: Job applications, printing, professional sharing

### HTML Export
1. Click **Export** → **Export Data (JSON)** - actually this exports as HTML
2. Browser downloads `[YourName]_Portfolio.html`
3. Open in any browser
4. Share via email or hosting platform

**Best for**: Sharing portfolios, self-hosting, long-term storage

### JSON Export
1. Click **Export** → **Export Data (JSON)**
2. Browser downloads `[YourName]_Portfolio.json`
3. Keep as backup of your customization
4. Share with team members

**Best for**: Backup, sharing configuration, re-importing later

## Troubleshooting

### Preview Not Loading?
- Refresh the page
- Check internet connection
- Try a different browser

### Changes Not Saving?
- Changes auto-save to browser
- Use Ctrl+Z if needed
- Export your portfolio regularly

### Export File Too Large?
- Consider using HTML export instead
- PDF export includes all styling

### Colors Look Different After Export?
- Check printer color settings
- PDF viewers may render colors differently
- Test in multiple viewers

### Profile Image Not Showing?
- Check file size (max 10MB)
- Ensure image format is JPG, PNG, or WebP
- Try re-uploading
