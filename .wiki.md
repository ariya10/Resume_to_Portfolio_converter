# Project Summary
The AI Resume-to-Portfolio Converter is a web application that transforms resumes (PDF/DOCX) into interactive portfolio websites. Leveraging AI technologies, it extracts structured data and enhances content, enabling users to create customizable portfolio templates. This tool is especially useful for students, fresh graduates, and professionals who aim to showcase their skills and experiences in a modern format.

# Project Module Description
- **Frontend**: Built with React, this module provides a user interface for uploading resumes, customizing portfolios, and exporting the final products.
- **Backend**: A Python FastAPI server that handles resume parsing, content generation, and API interactions with AI services.
- **AI Services**: Utilizes AI-driven content generation for processing resume data, enhancing the overall output.

# File Description Inventory
- **backend/**: Houses server-side code, including API routes, models, and services for handling resume parsing and data storage.
- **frontend/**: Contains client-side code, including components, pages, and configurations for the user interface.
- **lib/template-renderer.ts**: Library for rendering templates in the portfolio builder.
- **src/pages/Builder.tsx**: Updated page component for the portfolio builder with fullscreen preview functionality.
- **src/store/editor-store.ts**: State management file for handling editor actions and fullscreen state.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Python, FastAPI, Atoms Cloud for file storage and AI capabilities
- **AI**: DeepSeek for content generation and PDF analysis

# Usage
1. **Install Dependencies**:
   - For the backend: Run `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
   
2. **Build and Run**:
   - For the backend: Execute `uvicorn main:app --reload`
   - For the frontend: Execute `pnpm run build` and then `pnpm run serve` to start the application.
